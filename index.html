Simple Workflow

1

Create Tickets

Organize your work with tickets and acceptance criteria

hlavi tickets create "Feature name"
2

Configure Agent

Set up AI model and execution mode (attended/unattended)

hlavi agent configure
3

Let AI Work

Agent analyzes, plans, and executes your tasks

hlavi agent start HLA1
4

Review & Iterate

Approve changes or provide feedback for improvements

hlavi tickets show HLA1
